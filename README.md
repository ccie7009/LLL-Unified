# LLL-Unified
A tool to runs on a Lillygo T-Internet-POE ESP32, Waveshare ESP32-S3-ETH and Waveshare ESP32-P4-ETH micro controller and uses the ESP-IDF framework within PlatformIO

It will show the CDP, LLDP, EDP, FDP, NDP and MNDP information on a display which is an ST-7789P3 (76x284) using LVGL

There is a provision for a button to change the scroll speed

I have not tested the EDP, FDP or NDP protocols since I do not have the hardware to do testing

Use the select target in PIO to compile for the different micro controllers
